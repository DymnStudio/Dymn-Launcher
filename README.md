## Key Features

* 🏠 Customizable home screen
* 📂 Folder support
* 🔍 Fast app search
* 🎨 Icon pack support
* 🌈 Custom accent color selection
* 🖼️ Built-in and custom wallpapers
* 👆 Gesture support
* 📱 Widgets support
* ⚙️ Interface personalization
* 🚀 Fast and lightweight performance
* 🧩 Built with Jetpack Compose
